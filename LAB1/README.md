# LAB1 操作说明

本实验程序基于 `freeglut`，运行后会打开一个图形窗口，用于演示图形对象、顶点颜色和位置变化。

## 程序功能

- 显示正方形和圆形两种图形
- 支持单色填充和顶点颜色插值
- 支持用方向键移动图形在窗口中的位置

## 按键说明

- `1`：切换为正方形
- `2`：切换为圆形
- `c` / `C`：切换颜色模式
- 方向键：移动图形
- `r` / `R`：恢复初始状态
- `Esc`：退出程序

## 运行方式

### macOS

如果已经在本地编译出可执行文件，可以直接运行：

```bash
./main
```

如果需要重新编译：

```bash
clang++ -std=c++11 main.cpp -framework OpenGL -framework GLUT -o main
./main
```

### Windows

如果通过 GitHub Actions 下载了构建产物：

1. 下载 `windows-main` artifact
2. 解压压缩包
3. 双击 `main.exe` 运行

当前 workflow 目标是生成单文件 `main.exe`，不再额外附带 `freeglut.dll`。

## 演示内容对应关系

- 图形对象：通过 `1` 和 `2` 切换正方形、圆形
- 对象属性：通过 `c` 切换单色和顶点渐变色
- 观察方式：通过方向键改变图形在屏幕中的位置
